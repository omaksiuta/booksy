Service

  1. Service

    Service is most important part of BookMySlot. Let's get started with how to create and what you need while creating service. Below point you have to consider while creating new service.

    1) Title : This is a value to service title which you want to add.

    2) Select Service Category* : Select category that represent service.

    3) Select Staff : You can assign staff member to you serice If you have staff. You don't need to be worry if you are running businnes by single handed.

    4) Select City* : Select city where service is provided.

    5) Select Location* : Select location where service is provided.

    6) Address* : Add venue/address of service.

    7) Get Direction Map Link : You can add map link to get direction of service location.

    8) Description* : Add description about service.

    9) Select Days* : Select your working days for service. If you provide service on single day you can select single day.

    10) Slot Time * ( In Min. ) : Slot time is total amount of time require for that service, you have to add value in minute.

    11) Start Time (In 24hr format)* : Select start time of service from like 8:00 (morning).

    12) End Time (In 24hr format)* : Selecr end time of service. You have to enter value in 24 hr format only.

    13) Padding Time ( In Min. ) : You can add padding time between two consecutive time slot. Like if your first slot is of 45 min then if you add 15 min as a padding time then your next slot will be generate by adding 15 min.

    14) Do You want to allow multiple booking on same slot? * : You can allowe customer to book same slot multiple time. If you select yes it will ask you "Multiple Slot Booking Limit" so you have to add total number of user can book same slot.

    15) Type * : Service can be of two type free and paid.

    • Free
    • Paid*

      - Price *: Price that you want to set for service.

      - Discount in Percentage: If you want to add discount of service then you can add here.

      - From Date: You can add discount start date from where amount of discount will be consider.

      - To Date: You can add end date for discount. After that discount will not be applied on service.

    16) Service Image * : Select images represent your service.

    17) Service FAQs : If you have list of FAQ, then you can add it from here.

    18) SEO : You can add SEO details like Keyword,Description,Sog Image for that particular service.

Event

  1. Event

    From manage event page admin can able to create new Event and able to manage existing events. Below point you have to consider while creating new event

    1) Title*: Set event title that you want to add.

    2) Select Event Category*: Select cateory of event. Like Business, Meetup, Party etc.

    3) From Date: Select date from where event will be begin.

    4) To Date: Select date when event will be expired.

    5) Select City*: Select city of event.

    6) Select Location*: Select location of event.

    7) Venue*: Add venue of event.

    8) Get Direction Map Link: You can add direction map link here.

    9) Description*: Add description of your event, so user can get to know more about event.

    10) Ticket*: In ticket section you can add multiple ticket type.

    - You can create diffent ticket with different price.

    - One added ticket will be booked then given ticket type will not be show and can not available for booking.

    11) Media*: Upload media that shows and describe about your event.

    12) Sponsor Information: You can add sponsor for that particular evemt

    13) SEO: You can add SEO Keyword, description, OG Images for event.

    14) FAQs: If you like to add FAQs for event you can do that from here.

Report

Reports section is useful for graphical visualization of information

  1. 1) Vendor Report

    Using Vendor View report admin will quickly review the statistics about how many Vendor created on a daily basis over selected month.

  2. 2) Customer Report

    Using Customer report admin will quickly review the statistics about how many new customer will be created on daily basis over selected month.

  3. 3) Appointment Report

    Using Appointment report admin will quickly review the statistics about how many new appointments will be created by customer on daily basis over selected month.
