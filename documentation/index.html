Installation

This section will help you how to setup BookMySlot System.

The BookMySlot application takes seconds to set-up process using easy 4 following steps

  • System Requirement
  • Configuration
  • Site Setting
  • Email Setting
  1. Step 1 : System Requirement.

    This step is check basic system requirements for install BookMySlot System.

  2. Step 2 : Configuration.

    This step divided in 2 parts 1)Database Connection Details. 2)Administration Account Details.

    1)Database Connection Details : Provide Hostname, Username, Password and Database name.Using this detail establish the connection with database

    2)Administration Account Details : Provide your Admin account detail.Using this detail your Administration account will be created.

    Note: Please remember your Administration Account Details.Using this detail (Email / Password) you can access your administration account
  3. Step 3 : Site Setting.

    Using this step you can setup your site detail like as site name, email and logo that you want to setup

  4. Step 4 : Email Setting.

    Using this step you can setup your email detail like as SMTP Host, Username, Password, Port and SMTP Secure
