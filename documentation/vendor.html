Vendor Dashboard

  1. Vendor Dashboard.

    After login successfully Vendor dashboard come up with statistics about all module as well as vendor can perform urgent actions quickly

    Vendor dashboard display statistics of Event, Appointment, Total Earnings.

    On Dashboard vendor can able to see Pending service appointment/Event booking for approval.

My Staff

  1. My Staff.

    Vendor can add/update/delete staff member.

    Staff can be assign to service and customer can select staff member.

Wallet

  1. Wallet.

    ON wallet, page vendor can see withdrawable amount and based on that can create new request for payout.

    Payout request will be goes to admin and once admin process that vendor will get notification.

Gallery

  1. Gallery

    From manage gallery page vendor can able to upload new gallery image and able to update existing gallery images. Active gallery image will be displayed on front side vendor profile.

Inquiry

  1. Service & Vendor Inquiry

    Vendor have event inquiry module. Which display event inquiry got from customer for specific service/event.

Report

City & Location

  1. City & Location

    From manage city page vendor can able to create new City and view all existing City.

    Vendor only can able to create city and location if it is granted by vendor.
